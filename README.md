
  # DIGIRINGO

  This is the code bundle for DIGIRINGO — a virtual phone-number app (numbers, SMS, calls, inboxes) with a web Control Hub. Original UI design: https://www.figma.com/design/f1ft0bJSfkDImdvoJwy6PV/NumBox-Mobile-App-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
